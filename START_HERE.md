## Quick Start (Today)

### Step 1: Setup Frontend
```bash
cd TaskManagement.Frontend
npm install
```
⏱️ Takes ~2-3 minutes

### Step 2: Start Frontend
```bash
npm start
```
✅ Runs on http://localhost:4200

### Step 3: Login
1. Go to http://localhost:4200/login
2. Paste JWT token (with `access_api` scope)
3. Click "Sign In"
4. ✅ You're in!

### Step 4: Explore
- **Dashboard** - See your tasks overview
- **Tasks** - View and manage tasks
- **NLP Query** - Ask natural language questions
- **Time Tracking** - Track work hours
- **Projects** - Manage projects
- **Users** - View team members

## Example Workflows

### Workflow 1: Track Task Time
```
1. Login with JWT token
2. Click Dashboard
3. Click task you're working on
4. Click "Start Work"
5. Work on task
6. Click "Stop Work"
7. See time tracked
```

### Workflow 2: Ask NLP Questions
```
1. Go to NLP Query page
2. Type: "What did I do this week?"
3. See results with your tasks
4. Or ask: "How much time on ProjectX?"
5. Get time tracking summary
```

### Workflow 3: Generate Report
```
1. Go to Users page
2. Click user
3. See work logs
4. View time per task
5. Export or print report
```
